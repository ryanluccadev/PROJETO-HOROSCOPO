# 🔮 Projeto Horóscopo

Aplicação desenvolvida em **C** que identifica o signo do usuário a partir da data de nascimento informada. O programa realiza a validação da data, determina o signo correspondente e exibe a imagem do respectivo signo.

---

## ✨ Funcionalidades

* ✅ Validação da data de nascimento.
* ♈ Identificação automática do signo do zodíaco.
* 🖼️ Exibição da imagem correspondente ao signo.
* 💻 Interface simples executada em ambiente desktop.

---

## 🛠️ Tecnologias Utilizadas

* Linguagem C
* Biblioteca padrão da linguagem C
